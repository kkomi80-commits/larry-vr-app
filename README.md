# 라오어 VR 리밸런싱 Pro 📊

SOLX & TQQQ 포트폴리오를 위한 라오어 밸류 리밸런싱 PWA 앱 - **실시간 주가 연동 & 알림 기능**

## 🆕 Pro 버전 신기능

### ⚡ 실시간 주가 업데이트
- Alpha Vantage API 연동으로 실시간 주가 자동 업데이트
- SOLX, TQQQ 가격 변동률 표시
- 1분마다 자동 새로고침 옵션

### 🔔 리밸런싱 알림
- 목표 V값 이탈 시 자동 알림
- 매수/매도 타이밍 푸시 알림
- 알림 히스토리 보기

### 📱 두 가지 업데이트 방식
1. **API 자동 업데이트**: 무료 API 키로 자동 가격 업데이트
2. **수동 입력**: API 없이 직접 현재가 입력

## 📱 아이폰 설치 방법

### 1️⃣ GitHub Pages로 호스팅

1. [GitHub.com](https://github.com) 회원가입/로그인
2. New repository 생성
   - 이름: `larry-vr-pro`
   - Public 선택
3. 파일 업로드:
   - `index-pro.html` → `index.html`로 이름 변경하여 업로드
   - `manifest.json`
   - `sw.js`
4. Settings → Pages → Source를 "main"으로 설정
5. URL 생성 확인 (예: `https://yourname.github.io/larry-vr-pro/`)

### 2️⃣ 아이폰에서 설치

1. Safari에서 생성된 URL 접속
2. 화면 하단 **공유 버튼** (□↑) 탭
3. **"홈 화면에 추가"** 선택
4. "라오어 VR Pro" 확인 후 **추가**
5. 홈 화면에 앱 아이콘 생성 완료! ✅

## 🔑 API 키 발급 및 설정

### Alpha Vantage API (무료)

1. [Alpha Vantage 웹사이트](https://www.alphavantage.co/support/#api-key) 접속
2. 이메일 입력하고 **GET FREE API KEY** 클릭
3. API 키 복사 (예: `ABCD1234EFGH5678`)
4. 앱 실행 후 **"API 키 설정"** 버튼 클릭
5. 복사한 API 키 붙여넣기 후 **저장**

**무료 제한사항:**
- 분당 5회 호출
- 일일 500회 호출
- 충분한 개인 사용 가능!

### API 없이 사용하기

API 키 없이도 **"수동 입력"** 버튼으로 현재가를 직접 입력하여 사용 가능합니다.

## ✨ 주요 기능

### 1. 라오어 VR 공식
```
V = P + (P / G)
```
- **V**: 목표 자산 가치
- **P**: 풀(예수금)
- **G**: 기울기 (적립식 10, 인출식 20)

### 2. 실시간 주가 기능
- ✅ SOLX, TQQQ 실시간 가격 조회
- ✅ 가격 변동률(%) 표시
- ✅ 자동 새로고침 (1분 간격)
- ✅ 마지막 업데이트 시간 표시

### 3. 스마트 알림
- ✅ V값 임계값 이탈 시 자동 알림
- ✅ 매수/매도 타이밍 알림
- ✅ 브라우저 푸시 알림 지원
- ✅ 알림 히스토리 보기

### 4. 리밸런싱 분석
- ✅ 목표 V 대비 현재 자산 비율
- ✅ 정확한 매수/매도 수량 계산
- ✅ 리밸런싱 후 예상 풀 잔액
- ✅ 실시간 포트폴리오 분석

### 5. 투자 기록
- ✅ 시점별 포트폴리오 스냅샷
- ✅ V값, G값, 리밸런싱 내역
- ✅ 로컬 저장소에 안전 보관

## 🎯 사용 가이드

### 초기 설정

1. **API 키 설정** (선택사항)
   - 실시간 자동 업데이트를 원하면 설정
   - 수동 입력만 사용할 경우 건너뛰기

2. **VR 설정**
   - 투자 유형 선택
   - G값 확인/조정
   - 풀(P) 금액 입력

3. **목표 비율**
   - SOLX와 TQQQ 비율 설정
   - 리밸런싱 임계값 설정 (기본 5%)

4. **보유 수량 입력**
   - 각 자산의 보유 주식 수 입력

### 일상 사용

1. **주가 업데이트**
   - "API 업데이트" 버튼: 자동으로 최신 가격 조회
   - "수동 입력" 버튼: 직접 현재가 입력
   - 자동 새로고침: 1분마다 자동 업데이트

2. **리밸런싱 확인**
   - V 비율이 임계값 벗어나면 알림 발생
   - 리밸런싱 계획에서 매수/매도 수량 확인

3. **알림 활성화**
   - "리밸런싱 알림" 섹션에서 "활성화" 클릭
   - 브라우저 알림 권한 허용
   - 리밸런싱 필요 시 자동으로 알림 수신

4. **기록 저장**
   - 중요한 시점에 "현재 상태 저장" 클릭
   - 기록 버튼으로 과거 내역 확인

## 💡 사용 팁

### 효율적인 API 사용
- 무료 API는 일일 500회 제한
- 자동 새로고침은 필요할 때만 켜기
- 장 마감 후에는 수동 입력 활용

### 알림 최적화
- 임계값을 너무 낮게 설정하면 알림이 너무 자주 옴
- 5~10% 사이가 적당
- 알림 히스토리로 패턴 파악

### G값 실험
- 보수적: G = 15~20
- 중립적: G = 10
- 공격적: G = 5~8
- 본인 성향에 맞게 조정

## 🐛 문제 해결

### API 호출이 실패해요
- API 키가 올바른지 확인
- 일일 500회 제한 확인
- 네트워크 연결 확인

### 알림이 안 와요
- 브라우저 알림 권한 확인
- Safari 설정 → 알림에서 허용 확인
- 임계값 설정 확인

### 자동 새로고침이 안 돼요
- 앱이 백그라운드에 있으면 중단될 수 있음
- 앱을 포그라운드로 가져오기
- 수동 업데이트 버튼 사용
